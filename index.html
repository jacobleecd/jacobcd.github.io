I recently built Mashr, an open source data pipeline orchestration framework built with Node.js & Google Cloud Platform Services.

Mashr is an open source data pipeline orchestration and monitoring framework for small applications.

Mashr is built with Node.js, provides an easy-to-use CLI, uses Docker to host Embulk on GCE Instances on Google Cloud Platform (GCP) and is optimized for data pipeline best practices including monitoring, archiving & backup strategy in case of failover.
